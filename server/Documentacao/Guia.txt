dependências necessárias:

express: Nosso framework de servidor.

MARIADB: O "driver" oficial para conectar o Node.js ao MariaDB.

CORS: Para permitir que o seu front-end (em localhost:5173, por exemplo) se comunique com o seu back-end (em localhost:3001, por exemplo).

BCRYPTJS: Essencial para nunca salvar senhas em texto puro. Ele cria um "hash" da senha.

JSONWEBTOKEN: Usado para criar "tokens" de sessão após o login.

DOTENV: Para gerenciar variáveis de ambiente (como senhas de banco de dados).

NODEMON (Opcional, mas recomendado): Reinicia o servidor automaticamente quando você salva um arquivo.
